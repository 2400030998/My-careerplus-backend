 MyCareer+ Backend – Spring Boot Career Assessment API

MyCareer+ Backend is the server-side application of the MyCareer+ Web-Based Career Assessment Platform developed for the FSAD course.

It provides secure REST APIs for user login, OTP verification, assessments, and database management.

Backend Deployment Link:

https://your-railway-backend-link.up.railway.app/

Technologies Used

- Java
- Spring Boot
- MySQL
- Maven
- Twilio API
- Railway

Features

- User Authentication
- OTP Verification
- Assessment Management
- Database Connectivity
- Frontend Integration

## Conclusion

MyCareer+ Backend is the core part of the project that manages users, assessments, and system functionality efficiently.
